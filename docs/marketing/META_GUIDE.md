# Meta (Facebook + Instagram) guide

## Assets location

```
assets/marketing/output/
├── meta/feed/          # 1080×1080 flyers + captions
├── meta/stories/       # 1080×1920 story sequence (4 frames)
├── meta/link-ads/      # 1200×628 link preview images
```

## Instagram

### Feed posts (5 flyers ready)

| File | Best for |
|------|----------|
| `meta-feed-01.png` | General product intro |
| `meta-feed-02.png` | $200k hook |
| `meta-feed-03.png` | Google doc pain point |
| `meta-feed-04.png` | AI takeover angle |
| `meta-feed-05.png` | ATS platforms (Workday/Greenhouse/Lever) |

Captions in matching `*-caption.txt` files. **Include link in caption** (IG allows links in captions for business accounts, or use link in bio + "link in bio" CTA).

### Carousel posts (reuse TikTok slides)

Pick any `tiktok/accXX-postXX/slide-XX-square.png` series (6 slides). Carousels perform well for story-style content.

**Recommended carousels:**
- `acc01-post01` — $200k story (highest hook)
- `acc05-post03` — AI takeover
- `acc10-post03` — brand CTA

### Stories

Post `meta/stories/story-01.png` through `story-04.png` as a 4-frame story sequence. Add link sticker → `swiftdroom.com/register?ref=LAXJSLCA`.

## Facebook

Same flyers as IG feed. Facebook allows **direct links** in posts — always include:
```
https://www.swiftdroom.com/register?ref=LAXJSLCA
Code LAXJSLCA for 20% off
```
