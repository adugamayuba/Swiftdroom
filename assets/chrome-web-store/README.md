# Chrome Web Store assets

## Screenshots (1280×800)

Captured PNGs live in `screenshots/`:

| File | Scene |
|------|--------|
| `01-sidepanel-job-form.png` | Side panel on Greenhouse-style application |
| `02-autofill-progress.png` | Autofill stats + Ghostwriter on Workday-style form |
| `03-dashboard-settings.png` | Dashboard settings with extension connected |

Mocks used for capture: `mocks/*.html` (open at 1280×800 viewport).

Regenerate (recommended — exact 1280×800, no dark browser borders):

```bash
pip3 install playwright && python3 -m playwright install chromium
python3 scripts/capture-store-screenshots.py
```

## Logos

Generated by `scripts/generate-brand-assets.py` into `assets/logos/`:

- `styles/` — primary, lavender, mint, circle, wordmarks (512px / transparent)
- `sizes/` — 16, 48, 128, 256, 512 for store and extension
- `promo-small-440x280.png`, `promo-marquee-1400x560.png`

Extension icons are copied to `extension/icons/`.
