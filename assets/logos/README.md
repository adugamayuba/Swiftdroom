# Swiftdroom brand assets

Regenerate everything:

```bash
python3 scripts/generate-brand-assets.py
```

## Logo styles (`styles/`)

| File | Description |
|------|-------------|
| `icon-primary-512.png` | Dark teal square + lavender monogram (default) |
| `icon-lavender-512.png` | Lavender square + dark monogram |
| `icon-mint-512.png` | Mint square + dark monogram |
| `icon-circle-512.png` | Circular dark icon |
| `wordmark-dark-transparent.png` | Wordmark on transparent background |
| `wordmark-on-dark.png` | Wordmark on dark teal |
| `wordmark-on-lavender.png` | Wordmark on lavender |

## Sizes (`sizes/`)

- `icon-16.png`, `icon-48.png`, `icon-128.png`, `icon-256.png`, `icon-512.png` ‚Äî primary icon
- `store-icon-128.png` ‚Äî Chrome Web Store listing icon
- `lavender-16.png`, `lavender-48.png`, `lavender-128.png` ‚Äî alternate style
- `circle-16.png`, `circle-48.png`, `circle-128.png` ‚Äî circular style

## Promo

- `promo-small-440x280.png` ‚Äî small promo tile
- `promo-marquee-1400x560.png` ‚Äî marquee promo image

Extension icons are synced to `extension/icons/` on each run.
